# 📖 About

Feature Engineering is one of the most important steps in Machine Learning. Better features often lead to better-performing models.

This repository contains practical implementations of **feature engineering techniques**, **data preprocessing workflows**, and **statistical concepts** commonly used before training Machine Learning models.

Each notebook explains the intuition, mathematical background, implementation, visualization, and best practices for every technique.

# 🚀 Getting Started

Clone the repository

```bash

```

Navigate into the project

```bash
cd Feature-Engineering
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
